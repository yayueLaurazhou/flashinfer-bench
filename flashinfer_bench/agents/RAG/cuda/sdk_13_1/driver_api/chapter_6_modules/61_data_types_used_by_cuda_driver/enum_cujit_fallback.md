# enum CUjit_fallback

Cubin matching fallback strategies

###### Values

**CU_PREFER_PTX = 0**
Prefer to compile ptx if exact binary match not found
**CU_PREFER_BINARY**
Prefer to fall back to compatible binary code if exact match not found
