# __host__cudaError_t cudaGraphDestroy (cudaGraph_t graph)

Destroys a graph.

##### Parameters

**graph**

  - Graph to destroy

##### Returns

cudaSuccess, cudaErrorInvalidValue

##### Description

Destroys the graph specified by graph, as well as all of its nodes.
