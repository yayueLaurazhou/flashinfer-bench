# 9.7.1. Integer Arithmetic Instructions

### 9.7.1. [Integer Arithmetic Instructions](https://docs.nvidia.com/cuda/parallel-thread-execution/#integer-arithmetic-instructions)[ÔÉÅ](https://docs.nvidia.com/cuda/parallel-thread-execution/#integer-arithmetic-instructions "Permalink to this headline")

Integer arithmetic instructions operate on the integer types in register and constant immediate
forms. The integer arithmetic instructions are:

* `add`
* `sub`
* `mul`
* `mad`
* `mul24`
* `mad24`
* `sad`
* `div`
* `rem`
* `abs`
* `neg`
* `min`
* `max`
* `popc`
* `clz`
* `bfind`
* `fns`
* `brev`
* `bfe`
* `bfi`
* `bmsk`
* `szext`
* `dp4a`
* `dp2a`
