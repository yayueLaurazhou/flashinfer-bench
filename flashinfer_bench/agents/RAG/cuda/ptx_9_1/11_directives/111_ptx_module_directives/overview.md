# 11.1. PTX Module Directives

## 11.1. [PTX Module Directives](https://docs.nvidia.com/cuda/parallel-thread-execution/#ptx-module-directives)[ÔÉÅ](https://docs.nvidia.com/cuda/parallel-thread-execution/#ptx-module-directives "Permalink to this headline")

The following directives declare the PTX ISA version of the code in the module, the target
architecture for which the code was generated, and the size of addresses within the PTX module.

* `.version`
* `.target`
* `.address_size`
